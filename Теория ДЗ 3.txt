1. Исключение - Ошибка, которая может возникнуть внутри программного кода
2. Для того чтобы задать одинаковый разработчик для разных исключений необходимо указать нескколько классов в круглых скобках через запятую
3. Все исключения наследуются от класса BaseException
4. Сцепление исключений - совокупность исключений связанных в виде дерева
5. Синтаксическая ошибка - ошибка возникающая при неправильном написании кода (не хватает ':')
6. Предупреждение - Возможность избежать ненужных ошибок программы, которые не повлекут критической ошибки.
Сгенерировать предупреждение можно следующим кодом: warning.warn('сообщение',Stacklevel=int[0,10])
